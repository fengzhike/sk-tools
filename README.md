# sk-tools

create simple web app, with vue or react;

## Usage


```
npx sk-tools my-app

cd my-app

npm install

npm run dev

```

## cmd

```
npx sk-tools <projectName> [projectType]
```


## Support

| option | needed | type  | default  | value |
| -------| -------| -------| -----------| --------- |
| projectName   | True      | String |  ''  | eg: 'myapp'|
| prejectType   | False     | String |  ''  | [vue]  react or react-hook |
